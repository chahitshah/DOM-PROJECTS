# 🎥 Reels App (Day 30)

A simple Instagram Reels-inspired web app built using **HTML, SCSS, and JavaScript (DOM)**.
This project focuses on dynamic rendering, event handling, and interactive UI.

---

## 🚀 Features

* 📱 Vertical scroll reels (scroll-snap)
* ❤️ Like / Unlike functionality
* 💥 Double-click to like animation
* 👤 Follow / Unfollow button
* 🎬 Auto-playing videos
* 🧠 Dynamic content rendering using JavaScript
* 🎨 Clean UI with gradient and overlay effects

---

## 🛠 Tech Stack

* HTML
* SCSS (CSS)
* JavaScript (Vanilla DOM)

## ⚙️ How It Works

* Data is stored in a JavaScript array
* DOM elements are generated dynamically using `forEach`
* Event listeners handle:

  * Click (like toggle)
  * Double-click (heart animation ❤️)
  * Follow button toggle
