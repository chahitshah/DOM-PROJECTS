## 📂 Projects

### 🎥 Day 18 — Reels App

An Instagram Reels-inspired web app built using DOM manipulation.

#### ✨ Features

* Vertical scroll reels (scroll-snap)
* Double click ❤️ like animation
* Like / Unlike functionality
* Follow / Unfollow button

* Location-based time

* Animated cards
* Video backgrounds
* Multi-page navigation
